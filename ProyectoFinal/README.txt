Proyecto Final

Esté es el proyecto final de la asignatura de Programación I. Esté proyecto es muy sencillo: consiste en una aplicación que 
le permite a las personas crear figuras, tanto bidimensionales como tridimensionales. Les permite agregar información relevante de la figura,
como valor de las longitudes o el radio, dependiendo del tipo de figura. Luego, imprime información extra adiquirida gracias al input
dado por la persona. Esta información puede ser el area, el volúmen, el área superficial, etc. Al final, le pregunta al usuario si quisiera
salir del app o si quisiera crear otra figura diferente. 

En este proyecto se aplica el concepto de polimorfismo al crear una clase abstracta de figura, y otras dos subclases abstractas de los 
dos tipos de dimensiones. Luego de esto, se crean las clases principales. Me dediqué a crear 8 figuras en tota: 4 bidimensionales y las
otras 4 tridimensionales. Utilicé variables protected para facilitar el proceso de herencia, además de crear múltiples métodos Get. 

Al final, el proyecto funcionó como se esperaba. Aunque es sencillo, se utilizó lo aprendido en clases, desde POO hasta polimorfismo.
Posiblemente, en un futuro puediera expandir este proyecto hasta para poder imprimir las figuras pero hasta ahora, estoy conformé
con el producto actual.
